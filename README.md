# Three Scroll Demo

Base inicial para una demo de scroll controlada por una secuencia de imagenes.

## Arranque

```bash
npm install
npm run dev
```

## Secuencia actual

- Carpeta: `public/sequences/`
- Patron: `anim0001.webp` hasta `anim0100.webp`

## Siguiente iteracion

- Añadir textos y transiciones sincronizadas con el frame actual.
- Llevar la secuencia a un plano de `three.js` si queremos mezclarla con camara 3D.
- Sustituir la secuencia por una exportacion desde Blender o por varias secuencias encadenadas.
